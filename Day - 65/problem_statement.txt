Problem Solving for the day - 16.09.2025 - Corporate question 

Interleave the First Half and Second Half of a Queue

You are given a queue with an even number of integers. Your task is to rearrange the queue so that it interleaves the first half and second half of the queue.

Test Case 01 :

Input :

Queue = [1, 2, 3, 4, 5, 6]

Output :

[1, 4, 2, 5, 3, 6]


Test Case 02 :

Input :

Queue = [10, 20, 30, 40, 50, 60, 70, 80]

Output :

[10, 50, 20, 60, 30, 70, 40, 80]
