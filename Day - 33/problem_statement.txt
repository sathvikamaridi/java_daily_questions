A person is climbing a staircase. The staircase has n steps. The person can climb either 1 step or 2 steps at a time.

Write a program to count how many distinct ways the person can climb to the top of the staircase.

Implement your solution using Dynamic Programming in Java.

Test Case 1:
Input:
n = 1
Output:
1

Test Case 2:
Input:
n = 2
Output:	
2

Test Case 3:
Input:
n = 5
Output:
8
