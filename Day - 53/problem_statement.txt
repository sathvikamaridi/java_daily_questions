 Problem Solving for the day - 02.09.2025 - Corporate questions

You are given a list of intervals where each interval is represented as [start, end].Your task is to merge all overlapping intervals and return a list of non-overlapping intervals.

Test Case 01 :

Input :

arr = [[1,3],[2,6],[8,10],[15,18]]

Output :

[[1,6],[8,10],[15,18]]

Test Case 02 :

Input :

arr = [[1,4],[4,5]]

output :

[[1,5]]
